## CRUD GO + GORM + MYSQL 


## HOW TO START
```
#Install GO
Create Project
mkdir crud
cd crud
go mod init crud

#Install Echo FrameWork
go get github.com/labstack/echo/v4

#Install GORM Driver Mysql
go get gorm.io/driver/mysql

#Go run
go run main.go
```
